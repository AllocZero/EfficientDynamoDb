RegionEndpoint

You need to specify a region that contains your DB. There are static properties for every supported region in RegionEndpoint class.

List of supported regions:

  1. RegionEndpoint.USEast1 - US East (N. Virginia)
  2. RegionEndpoint.USEast2 - US East (Ohio)
  3. RegionEndpoint.USWest1 - US West (N. California)
  4. RegionEndpoint.USWest2 - US West (Oregon)
  5. RegionEndpoint.AFSouth1 - Africa (Cape Town)
  6. RegionEndpoint.APEast1 - Asia Pacific (Hong Kong)
  7. RegionEndpoint.APSouth1 - Asia Pacific (Mumbai)
  8. RegionEndpoint.APNorthEast1 - Asia Pacific (Tokyo)
  9. RegionEndpoint.APNorthEast2 - Asia Pacific (Seoul)
  10. RegionEndpoint.APNorthEast3 - Asia Pacific (Osaka-Local)
  11. RegionEndpoint.APSouthEast1 - Asia Pacific (Singapore)
  12. RegionEndpoint.APSouthEast2 - Asia Pacific (Sydney)
  13. RegionEndpoint.CACentral1 - Canada (Central)
  14. RegionEndpoint.CNNorth1 - China (Beijing)
  15. RegionEndpoint.CNNorthWest1 - China (Ningxia)
  16. RegionEndpoint.EUCenteral1 - Europe (Frankfurt)
  17. RegionEndpoint.EUWest1 - Europe (Ireland)
  18. RegionEndpoint.EUWest2 - Europe (London)
